# ns-server-template

webpack+typescript+koa2 node server template

## Available Scripts

In the project directory, you can run:

### `npm run lint`

进行 lint 检测

### `npm run lint-fix`

进行 lint 检测并修复 lint 错误

### `npm run build`

构建产物
